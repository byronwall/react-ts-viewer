### Fenced Code Block (JavaScript)

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
greet("World");
```

### Fenced Code Block (Python)

```python
def greet(name):
  print(f"Hello, {name}!")

greet("World")
```
